# MXFY (Make X for Y) Documentation

**Component Type:** Orchestration & Ethics Layer  
**Status:** Production

## Overview

MXFY (Make X for Y) provides intent parsing and template-based synthesis, enabling natural language intent specification and automatic workflow generation for complex multi-step tasks.

## Key Capabilities

- Natural language intent parsing
- Template-based workflow synthesis
- Goal decomposition and planning
- Context-aware task generation
- Integration with PCT for spatial reasoning
- SDL-backed semantic understanding
