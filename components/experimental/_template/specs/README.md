# Specifications

Place your component specifications here.

## Guidelines

- Use JSON or YAML format
- Follow STUNIR specification format
- Include clear descriptions
- Define input/output contracts
- Specify dependencies

## Example Structure

```
specs/
├── component-spec.json       # Main component spec
├── operations/               # Operation definitions
├── types/                    # Type definitions
└── index.json               # Spec catalog
```
