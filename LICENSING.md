## Why Dual Licensing?

- **Protects open-source community:** AGPLv3 ensures contributions remain open
- **Enables commercial use:** Commercial license allows proprietary integration
- **Prevents theft:** Strong copyleft prevents unauthorized proprietary use
- **Sustainable development:** Commercial licensing supports ongoing development

## Which License Should I Choose?

**Choose AGPLv3 if:**
- Your project is open-source
- You're comfortable with copyleft requirements
- You want to contribute back to the community

**Choose Commercial License if:**
- You're building proprietary software
- You cannot disclose source code
- You need to avoid copyleft obligations
- You want priority support
